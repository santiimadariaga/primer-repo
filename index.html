EXTINTORES A BASE DE POLVO

Matafuegos de 1kg, 3,5kg, 5kg y 10kg.

Diseñados para proteger áreas que contienen riesgos de fuego Clase A (combustibles sólidos), Clase B (combustibles líquidos y gaseosos) o Clase C (equipos eléctricos energizados).

EXTINTORES A BASE DE AGUA

Matafuegos de 3L y 5L.

Diseñados para proteger áreas que contienen riesgos de fuego Clase A (combustibles sólidos).

EXTINTORES A BASE DE CO2

Matafuegos de 3,5kg, 5kg y 10kg.

Diseñados para proteger áreas que contienen riesgos de incendio Clase B (combustibles líquidos y gaseosos) y Clase C (equipos eléctricos energizados).

EXTINTORES DE ACETATO DE POTASIO K

Matafuegos de 3,5kg y 5kg.

Diseñados especialmente para extinguir fuegos de aceites vegetales en freidoras de cocinas comerciales.

EXTINTORES A BASE DE HALOCLEAN

Matafuegos de 5kg.

Diseñados para proteger áreas que contienen riesgos de fuego Clase A (combustibles sólidos), Clase B (combustibles líquidos y gaseosos) y Clase C (equipos eléctricos energizados).
